# Commit Club Bot — MVP

## Быстрый старт

### 1. Получить токен
Написать @BotFather → /newbot → скопировать токен

### 2. Получить GROUP_ID
- Добавить бота в группу как администратора
- Написать что угодно в группу
- Открыть: https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
- Найти "chat":{"id": ...} — это и есть GROUP_ID (отрицательное число)

### 3. Получить свой ADMIN_ID
Написать @userinfobot — он пришлёт ваш user_id

### 4. Запуск

pip install -r requirements.txt

# Создать .env файл:
cp .env.example .env
# Отредактировать .env, вставить реальные значения

# Запуск:
export $(cat .env | xargs) && python bot.py

### 5. Деплой на Railway (бесплатно)
1. railway.app → New Project → Deploy from GitHub
2. Добавить переменные окружения в Settings → Variables
3. Start command: python bot.py

## Команды

### Участник
- /start — регистрация (цель + никнейм + анонимность)
- /profile — моя статистика
- /report — напоминание как сдать отчёт
- Просто текст или фото — засчитывается как отчёт
- /callout КОД — отметиться на созвоне

### Администратор
- /setcallout КОД — установить код дня перед созвоном
- /remind — разослать напоминания вручную
- /leaderboard — показать рейтинг прямо сейчас

## Механика XP
- +10 XP — любой отчёт
- +5 XP — отчёт с фото/доказательством
- +5 XP — ранний старт (до 12:00 UTC)
- +50 XP — 7-дневный стрик
- +20 XP — присутствие на созвоне

## Ранги (по накопленному XP)
- ⬛ E — старт
- 🟫 D — 100 XP
- 🟦 C — 300 XP
- 🟩 B — 700 XP
- 🟨 A — 1500 XP
- 🟥 S — 3000 XP

## Расписание автоматических событий (UTC)
- 14:00 UTC (20:00 KG) — напоминания тем, кто не сдал отчёт
- 16:00 UTC (22:00 KG) — рейтинг дня в группу
